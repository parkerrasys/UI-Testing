Overview

AutoGrid is a powerful tool for planning autonomous routines in VEX Robotics. This documentation explains the platform's capabilities, features, and how to effectively use then with your robot

Whether you're new to robotics or an experienced competitor, AutoGrid provides intuitive interfaces and powerful planning tools to enhance your autonomous programming workflow.

Installation / Usage

AutoGrid is a web-based application, eliminating the need for installation. Simply access the platform through your web browser to get started. This approach ensures compatibility across different operating systems and devices.

Web Access

Visit https://parkerrasys.github.io/AutoGrid-Planner to access the AutoGrid Planner.

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection
  • Screen resolution of at least 1024x768 (1920x1080 recommended)

Getting Started

This section covers the fundamentals of using AutoGrid for your robotics projects. Follow these steps to begin creating your autonomous routines quickly:

Step 1: Create a New Project

From the AutoGrid dashboard, click the New Project button and set a name for your path.

Step 2: Layout Your Path

Use the Field Editor to place points on the field.

Step 3: Export, Share, or Build

Use the three button on the bottom left of the screen to export, share, or build your code.

Step 4: Download the AutoGrid Competition Template

Paste your built path into the autonomous funtion in our Competition Template and upload.

Step 5: Adjust Values

Fine tune values of your robot such as size, wheel diameter, and gearing to accuratly calculate the built path.

How AutoGrid Works

Understanding these fundamental concepts will help you make the most of AutoGrid's capabilities:

Calculations

The robot calculates the angle and distance it needs to go by making a triangle out of the X and Y distances of the coordinate it is trying to reach before using those values to know where it needs to go.

Key Features

AutoGrid offers a comprehensive set of features designed to streamline your autonomous programming workflow:

Path Planning

Design complex robot movements with intuitive point-and-click interfaces. (Path optimization coming soon)

Simulation

Show a view of how the robot will act on the field by simulating the robots movements.

Code Generation

Automatically generate autonomous code for the AutoGrid Competition Template to use.

Commands Reference

The AutoGrid Competition Template has a few funtions to call in order to move the robot, these funtions can be used manually or generated based on path input from the AutoGrid Planner.

Command Description Usage Example
setStarting(x, y); Sets the starting position of the robot setStarting(0, 0)
moveTo(x, y); Moves the robot to the specified coordinates moveTo(1, 5)
lookAt(x, y) Rotates the robot to face a specific point lookAt(4, 3)

Tips & Best Practices

Expert advice to help you get the most out of AutoGrid:

Start Simple

Begin with basic routines and gradually add complexity as you become more familiar with the platform.

Test Incrementally

Deploy and test small portions of your autonomous routine before attempting to run the entire sequence.

Plan for Variability

Design your routines with some tolerance for positioning errors and field variations.

Optimize for Time

Experiment with multiple paths to see what works best with your robot. Timing may vary.
