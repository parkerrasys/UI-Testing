Common Issues

Why arn't my recent paths loading?

Your recent paths might not be loading becuase of an issue on your end, or you have too many paths. Try refreshing the tab or clearing the site data to fix the issue. (This would remove all recent paths)

Why cant I import a share code?

The share code might not be importing correctly because you accidentally typed in the code. Try asking your friend for a new link and dont type in it at all.

Do I need to use the AutoGrid Planner?

Not at all! Feel free to mess around with the funtions in the AutoGrid Competeition Template! Get your hands dirty and learn how AutoGrid works from the ground up!

Why I cant export, share, or build my path?

You arnt able to export, share, or build empty paths! Make sure you have a recent path that has a path drawn selected, then try exporting, sharing, or building again.

How do I use built code?

In order to use the built output, you must download the AutoGrid Competition Template from our github here. Then replace the default autonomous code with the built code and upload!
